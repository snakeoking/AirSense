# AirSense

An end-to-end **Climate & Air Quality Monitoring System** developed for a Data Science Hackathon. The project performs data cleaning (ETL), pollution alert detection using Data Structures, climate visualization, and an interactive AQI Risk Map, all integrated into a **Streamlit Web Application**.

---

## 📌 Features

- ✅ ETL Pipeline using Pandas
- ✅ AQI Calculation
- ✅ Missing Value Handling
- ✅ Pollution Alert System (Stack & Priority Queue)
- ✅ Climate Dashboard (Matplotlib)
- ✅ Interactive AQI Risk Map (Folium)
- ✅ Streamlit Web Application
- ✅ Download Cleaned Dataset

---

## 🛠️ Technologies Used

- Python
- Pandas

- Matplotlib

## 📊 Project Modules

### Task 1 – ETL Pipeline
- Clean missing values
- Remove sensor errors
- Normalize city names
- Calculate AQI
- Export cleaned dataset

### Task 2 – Pollution Alert System
- Stack Implementation
- Priority Queue Implementation
- Top 10 Pollution Alerts
- Three-Hour Critical Window Detection

### Task 3 – Climate Dashboard
- AQI Time Series
- Heatmap
- Grouped Bar Chart
- Pie Charts

### Task 4 – AQI Risk Map
- Interactive Folium Map
- Circle Markers
- HeatMap
- AQI Popups
- Feature Groups
